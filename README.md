# neon-cipher-text

An interactive text animation that decodes characters on hover with a neon cyberpunk theme.

## Features

- **Cyberpunk Aesthetic**: Neon Cyan and Pink colors with a dark gradient background.
- **Hacker Text Effect**: Text "decodes" from random characters (letters, numbers, symbols) into the final "CYBERPUNK" text.
- **Interactive**: The decoding effect triggers on mouse hover (`pointerenter`) and resets on leave.
- **Responsive**: Adjusts font sizes for smaller screens.

## Technologies Used

- **HTML5**: Structure.
- **CSS3**: Styling (Orbitron font, gradients, text-shadows, animations).
- **JavaScript**: Logic for the character scrambling effect using `setInterval`.

## How to Use

1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Hover over the text "CYBERPUNK" to see the effect in action.
